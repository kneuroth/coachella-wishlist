# Coachella Wishlist

A small API for me and my friends to rate Coachella artists and see where our tastes align. Built with Node.js + Express on AWS Lambda, backed by DynamoDB.

## Setup

```
npm install
serverless deploy
```

## API

### Users

**Create a user**
```
POST /users
{ "name": "Alice" }
```
Returns `{ "userId": "<auto-generated uuid>", "name": "Alice" }`.

**Get a user**
```
GET /users/:userId
```

---

### Wishlists

Rankings are one of: `must_see`, `would_like_to_see`, `would_skip`.

**Get all friends' wishlists** (main page data, one call)
```
GET /wishlists
```
Returns all entries grouped by userId:
```json
{
  "user-id-1": [{ "artistId": "beyonce-2025", "ranking": "must_see", "updatedAt": "..." }],
  "user-id-2": [...]
}
```

**Get one user's wishlist**
```
GET /wishlists/:userId
```

**Get all ratings for one artist** (alignment view)
```
GET /wishlists/artist/:artistId
```
Returns `[{ "userId", "ranking", "updatedAt" }]` for every friend who rated that artist.

**Add or update a ranking**
```
PUT /wishlists/:userId/:artistId
{ "ranking": "must_see" }
```

**Remove a ranking**
```
DELETE /wishlists/:userId/:artistId
```

**Clear all wishlist data**
```
DELETE /wishlists
```
